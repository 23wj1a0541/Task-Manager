# Task Manager App (GitHub Copilot Dev Days)

A simple hands-on task manager web app built with vanilla HTML, CSS, and JavaScript.

## Project files

- `index.html` - Page markup and UI structure
- `style.css` - Dark theme styling and responsive layout
- `app.js` - Task management logic (add, complete, delete, filter, localStorage)
- `LAB_GUIDE.md` - Step-by-step prompts to run in GitHub Copilot for the expected lab output

## Run locally

1. Open the folder in your file explorer:
   - `c:\User\Downloads\Task-Manager`
2. Double-click `index.html`, or open it in your browser (Chrome/Edge/Firefox).
3. The app should load immediately without any build step.

## Usage

- Add a task with title and optional priority (low/medium/high).
- Mark tasks complete (strikethrough + green check).
- Delete tasks (confirmation required).
- Filter tasks: All / Active / Completed.
- Task counter displays progress (X of Y completed).
- Tasks persist via `localStorage` across refreshes.

## Lab guide

Follow the prompts in `LAB_GUIDE.md` and the GitHub Copilot instructions for this lab to get the desired task manager behavior and output. The guide walks through each feature and verification step.
